# Global Pollution Analysis and Prediction Using Machine Learning

## Project Overview
This project analyzes pollution-related environmental factors and predicts AQI classification using supervised machine learning algorithms.

## Dataset
Global Pollution Dataset 2025–2026

## Target Variable
AQI_Class

## Machine Learning Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Results
Random Forest and Decision Tree achieved the highest accuracy.

## How to Run
1. Install required libraries:

```bash
pip install -r requirements.txt
```

2. Run Jupyter Notebook:

```bash
jupyter notebook
```
